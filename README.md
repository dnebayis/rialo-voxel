# Rialo Voxel Interface

Interactive 3D visualization of the Rialo Network ecosystem, built with React Three Fiber.

## Overview

This application visualizes the progression of the Rialo ecosystem through 4 interactive stages:
1.  **The Raw Real World**: Fragmented and chaotic data.
2.  **The Rialo Network**: Unification and ordering of assets.
3.  **Application Assembly**: Building strictly verified structures.
4.  **Connected Value**: A solid-state global economy.

## Stack

*   **Framework**: Next.js
*   **3D Engine**: Three.js / React-Three-Fiber
*   **Animations**: Framer Motion
*   **Styling**: Vanilla CSS

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
https://rialo-voxel.vercel.app/
